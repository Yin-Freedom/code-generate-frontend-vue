# code-generate-frontend-vue

## 1. 开发规范

代码格式化：使用 eslint + prettier 语法检查和格式化，使用 vscode 插件在保存时自动生效。

配置：

1. 配置文件
   |-- .vscode
   | |-- extensions.json
   | |-- settings.json
   |--
   |-- .eslintrc.js
   |-- .prettierrc.js
2. 安装 vscode 插件
